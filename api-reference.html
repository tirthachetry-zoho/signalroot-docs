Base URL

Core Endpoints

Health Check

GET /actuator/health

Check the health status of the SignalRoot service.

Response:

{
  "status": "UP",
  "components": {
    "db": {
      "status": "UP",
      "details": {
        "database": "PostgreSQL",
        "validationQuery": "isValid()"
      }
    },
    "ping": {
      "status": "UP"
    }
  }
}

Alert Management

POST /api/alerts/process

Process and enrich incoming alerts from monitoring systems.

GET /api/alerts

Retrieve processed alerts with filters.

PUT /api/alerts/{id}

Update alert status or add notes.

Team Management

GET /api/teams

Get list of all teams.

POST /api/teams

Create a new team with configuration.

Webhook Management

GET /api/webhooks

List configured webhooks.

POST /api/webhooks

Configure new webhook endpoints.

SDK & Libraries

JavaScript/TypeScript

npm install @signalroot/sdk

Modern JavaScript SDK with TypeScript support.

Python

pip install signalroot-sdk

Python SDK for integration and automation.

Error Codes

Code Meaning Solution
400 Bad Request Check request format and parameters
401 Unauthorized Check API key authentication
404 Not Found Verify endpoint URL
429 Rate Limited Reduce request frequency
500 Internal Error Contact support if persistent
