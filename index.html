Quick Start Guide

Prerequisites

Before you begin, ensure you have the following installed:

  • Node.js (version 16 or higher)
  • Java (version 17 or higher)
  • Maven (version 3.6 or higher)
  • Docker (recommended for database)

Docker Quick Start (Recommended)

The fastest way to get SignalRoot running is with Docker:

# Clone the repository
git clone https://github.com/signalroot/signalroot.git
cd signalroot

# Start all services
docker-compose up -d

# Wait for services to be ready (about 2 minutes)
docker-compose logs -f

This will start:

  • Backend API on http://localhost:8080
  • Frontend Dashboard on http://localhost:3000
  • PostgreSQL Database on port 5432
  • Redis Cache on port 6379

Verify Installation

Check that everything is working:

# Check backend health
curl http://localhost:8080/actuator/health

# Expected response:
{"status":"UP"}

API Overview

Base URL

Production: https://api.signalroot.com
Development: http://localhost:8080

Authentication

SignalRoot uses API key authentication for secure access.

curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.signalroot.com/api/teams

Core Endpoints

Health Check

GET /actuator/health

Check the health status of the SignalRoot service.

Teams Management

GET /api/fake-customers/teams

Get all teams with their configurations.

Alert Processing

POST /webhooks/alerts/pagerduty

Process PagerDuty alerts with enrichment.
